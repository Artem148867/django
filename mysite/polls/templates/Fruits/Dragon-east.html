Dragon Fruit (East)

The Dragon Fruit allows the user to gain the power, and transform to a Mythical Dragon. The user can transform either to an Eastern Dragon; resembling those from the Chinese culture and the user can also transform into a large, tall human-hybrid dragon that deals more massive fiery damage.

Hybrid form

In Hybrid form, the user gains a 20% damage resistance to all attacks, causes a burning effect with M1 attacks, and has their moveset greatly enhanced.

Dragon form

In Dragon form, the user gains a 53% damage resistance to all attacks and is able to fly. The user can still be affected by crowd control but still able to move.
